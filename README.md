# צ'אט AI — מטלה 1 קורס סוכני AI

## תיאור הפרויקט
אפליקציית צ'אט המתקשרת עם מודל שפה GPT-5-NANO דרך API של המכללה, עם ממשק ויזואלי ב-Streamlit.

## דרישות מערכת
- Python 3.x
- חיבור לאינטרנט
- מפתח API מהמכללה

## הוראות התקנה
```bash
pip install streamlit requests
```

## הגדרת מפתח API
החליפי את `YOUR_API_KEY_HERE` בקובץ `app.py` ו-`chat.py` במפתח שקיבלת מהמכללה.

## הוראות הפעלה

### הרצת צ'אט CLI:
```bash
python chat.py
```

### הרצת ממשק Streamlit:
```bash
streamlit run app.py
```
