# 🤖 סוכן AI חכם — מטלה 1 קורס סוכני AI

ברוכות הבאות לפרויקט סוכן ה-AI של ליטל ואושיק!
הפרויקט מציג צ'אט חכם המסוגל לחפש מידע בזמן אמת ברשת ולנהל היסטוריית שיחה.

## קבצים בפרויקט

- `chat.py` — לולאת צ'אט בסיסית בטרמינל
- `app.py` — ממשק ויזואלי מתקדם ב-Streamlit עם חיפוש ברשת

## התקנה

```bash
pip install requests streamlit
```

## הרצה בטרמינל

```bash
python chat.py
```

## הרצה בממשק ויזואלי

```bash
streamlit run app.py
```

## תכונות מרכזיות

- **Web Search** — המודל מחפש מידע עדכני באינטרנט בזמן אמת
- **ניהול היסטוריה** — שמירת הקשר שיחה מלא בין הודעות
- **ניקוי תשובות** — הסרת לינקים ומקורות מיותרים מהתשובה
- **מעקב טוקנים** — תצוגת כמות הטוקנים שנוצלו היום
